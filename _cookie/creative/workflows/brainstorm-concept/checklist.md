# Brainstorm Concept Checklist

## Brief Intake
- [ ] Brief or topic received from user
- [ ] Target audience identified
- [ ] Brand context loaded (if brand config exists)
- [ ] Constraints clarified (duration, format, budget)
- [ ] Previous episode concepts checked to avoid repetition

## Creative Data
- [ ] Cinematic styles data loaded
- [ ] Color palettes data loaded
- [ ] Music mood mapping data loaded
- [ ] Transition library data loaded
- [ ] Color theory skill consulted
- [ ] Existing style guide loaded (if applicable)

## Concept Generation
- [ ] 3-5 distinct concepts generated
- [ ] Concepts are genuinely different approaches (not variations)
- [ ] Each concept has: title, hook, narrative arc
- [ ] Each concept has: visual style, color palette, lighting
- [ ] Each concept has: music mood (genre, tempo, energy)
- [ ] Each concept has: target format recommendation
- [ ] Each concept has: estimated duration
- [ ] Each concept has: estimated cost range
- [ ] Each concept has: pros and cons

## Presentation and Selection
- [ ] Concepts presented in comparable format
- [ ] Tradeoffs highlighted (creative vs safe, cheap vs quality)
- [ ] User selected a concept (or requested refinement)
- [ ] Feedback incorporated into selected concept

## Output
- [ ] Selected concept refined and expanded
- [ ] Narrative arc expanded to scene-level outline
- [ ] Visual style, music, and format confirmed
- [ ] Concept document output for idea-to-script workflow
- [ ] Concept selection saved to creative director sidecar
