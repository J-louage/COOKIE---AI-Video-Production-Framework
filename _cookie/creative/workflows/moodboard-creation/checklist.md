# Moodboard Creation Checklist

## Context Loading
- [ ] Selected concept or script loaded
- [ ] Brand config loaded (if exists)
- [ ] Existing style guide loaded (if applicable)
- [ ] Color palettes and cinematic styles data loaded

## Color Palette
- [ ] 5-7 colors selected with hex codes
- [ ] Each color has usage notes (primary, secondary, accent, background, text)
- [ ] Each color has emotional association annotation
- [ ] Accessibility contrast ratios verified for text colors
- [ ] Palette aligns with brand config (if exists)

## Typography
- [ ] Display/heading font selected
- [ ] Body/caption font selected
- [ ] Size hierarchy defined (title, subtitle, body, caption)
- [ ] Fonts available in Remotion (Google Fonts or bundled)
- [ ] Typography aligns with brand voice

## Reference Images
- [ ] Cost gate approved for image generation
- [ ] 2 establishing/environment shots generated
- [ ] 2 character mood/pose references generated
- [ ] 2 detail/texture references generated
- [ ] 1-2 title card/text overlay references generated
- [ ] Each image annotated with purpose
- [ ] Images generated at 1K resolution (cost-efficient)

## Style Direction
- [ ] Lighting direction defined (key light, color temp, ambient, shadows)
- [ ] Camera style defined (movement, lens, depth of field)
- [ ] Music/audio direction defined (genre, instruments, tempo, energy, mood)
- [ ] Transition style defined (preferred transitions from library)

## Approval and Output
- [ ] Complete moodboard compiled into structured document
- [ ] Moodboard presented to user for review
- [ ] User feedback incorporated
- [ ] Moodboard approved
- [ ] Moodboard saved to episode directory
- [ ] Moodboard output for create-style-guide workflow
- [ ] Decisions saved to creative director sidecar
