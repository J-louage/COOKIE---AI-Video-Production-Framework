# Brand Alignment Review Checklist

## Brand Context
- [ ] Brand config loaded from characters/brand/
- [ ] Style guide loaded (if exists)
- [ ] Brand colors and fonts identified
- [ ] Previous review scores loaded from sidecar (if any)
- [ ] Asset type identified (script, SSD, export, or moodboard)

## Review Asset
- [ ] Asset loaded and key elements extracted
- [ ] Asset type appropriate for brand review

## Tone & Voice (1-5)
- [ ] Narration matches brand voice guidelines
- [ ] Messaging consistent with brand positioning
- [ ] Emotional tone aligns with brand identity
- [ ] Language reflects target audience
- [ ] Specific references provided for scores below 4

## Visual Identity (1-5)
- [ ] Brand colors used correctly
- [ ] Typography consistent with brand fonts
- [ ] Logo placement correct (if applicable)
- [ ] Visual elements match style guide
- [ ] Overall visual feel aligned with brand mood
- [ ] Specific references provided for scores below 4

## Messaging & Content (1-5)
- [ ] Content serves brand objectives
- [ ] CTA clear and brand-appropriate
- [ ] Hook engages target audience
- [ ] Narrative coherent and purposeful
- [ ] No contradictions to brand messaging
- [ ] Specific references provided for scores below 4

## Audience Alignment (1-5)
- [ ] Content appropriate for target demographic
- [ ] Pacing matches platform expectations
- [ ] Accessibility standards met (captions, contrast, text size)
- [ ] Content length matches platform norms
- [ ] Specific references provided for scores below 4

## Scoring and Gate
- [ ] Overall score calculated (average of four categories)
- [ ] Approval status determined (approved / conditional / rework)
- [ ] Critical flags raised for any category below 3.0
- [ ] Recommendations prioritized by impact
- [ ] Minimum changes listed (if conditional)
- [ ] Rework workflow identified (if needs rework)
- [ ] Review saved to creative director sidecar
