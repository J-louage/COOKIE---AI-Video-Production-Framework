# Pre-Production Review Checklist

## Required Checks
- [ ] SSD (scene-spec.yaml) loaded and present
- [ ] All character identity files loaded and present
- [ ] Style guide loaded and present
- [ ] Dependency graph loaded and present
- [ ] Prompt library loaded and present
- [ ] Budget configuration loaded
- [ ] T6 SSD validation executed and passed
- [ ] T7 character validation executed for all characters and passed
- [ ] No missing pre-production artifacts

## Quality Standards
- [ ] Narrative flow reviewed and approved (scene transitions, story arc, hook)
- [ ] Pacing assessed and appropriate for format
- [ ] Character usage is purposeful in every scene
- [ ] Visual directions support narrative intent
- [ ] Total estimated cost within budget limits
- [ ] Most expensive scenes justified
- [ ] Cost optimization opportunities evaluated
- [ ] Re-generation buffer is adequate
- [ ] Dependency graph has no unresolved dependencies
- [ ] Prompt library covers every scene and clip

## Output Verification
- [ ] Pre-production summary dashboard compiled
- [ ] Director review notes documented
- [ ] Producer cost review notes documented
- [ ] Gate decision recorded (approved or not approved)
- [ ] Decision includes timestamp and reviewer identity
- [ ] If not approved: specific iteration instructions provided
- [ ] If not approved: relevant pre-production workflow identified for revision
- [ ] If approved: any non-blocking notes attached as conditions
