# Idea to Script Checklist

## Required Checks
- [ ] Script follows the canonical format template with all required sections
- [ ] Header includes title, episode ID, format type, target duration, and character list
- [ ] Every scene has a unique sequential scene ID
- [ ] Every scene has an estimated duration in seconds
- [ ] Every scene has a setting/environment description
- [ ] Every scene lists all characters present
- [ ] Every scene has detailed visual direction (camera, lighting, action)
- [ ] Every scene has detailed audio direction (narration, music, SFX)
- [ ] Total duration is within 10% of the target duration from the format preset
- [ ] All character names are valid and match entries in the characters directory
- [ ] Character references use [character:X] placeholder syntax

## Quality Standards
- [ ] Narration word count is feasible for each scene's allocated duration
- [ ] Music directions are specific enough for sourcing (mood, tempo, genre)
- [ ] SFX directions include timing cues and descriptive labels
- [ ] Visual directions specify camera movement (pan, zoom, static, etc.)
- [ ] Lighting and color palette notes are consistent with style guide
- [ ] Scene transitions are specified (cut, fade, dissolve, etc.)
- [ ] Hook scene appears within first 3 seconds for short-form content
- [ ] Pacing follows the format preset guidelines

## Output Verification
- [ ] Canonical script saved to episodes/{episode_id}/script/canonical-script.md
- [ ] Draft backup saved with timestamp
- [ ] Directory structure created if it did not exist
- [ ] Format preset was loaded and applied correctly
