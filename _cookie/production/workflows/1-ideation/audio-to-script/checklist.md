# Audio to Script Checklist

## Required Checks
- [ ] Audio file received or transcript provided
- [ ] Transcription completed with word-level timestamps
- [ ] Speaker identification applied where multiple voices detected
- [ ] Pauses, emphasis, and tone shifts annotated
- [ ] Content analyzed for topic breaks and narrative arc
- [ ] Visual scenes generated for every narration segment
- [ ] Scenes time-aligned to narration timestamps
- [ ] Scene transitions align with natural pauses or topic shifts
- [ ] Canonical script assembled with pre-filled narration text

## Quality Standards
- [ ] Transcription accuracy verified against source audio
- [ ] Visual directions complement narration content (not merely illustrate)
- [ ] No scenes cut mid-sentence or mid-thought
- [ ] Scene types appropriate for content (b-roll, character, screen recording, etc.)
- [ ] Camera movements specified for all visual scenes
- [ ] Music and SFX suggestions included in audio direction
- [ ] Narration pace does not exceed 160 wpm in any scene
- [ ] Visual complexity feasible within allocated scene duration

## Output Verification
- [ ] Tight timing conflicts flagged with specific recommendations
- [ ] Total duration matches source audio duration
- [ ] Script follows canonical format template with all required sections
- [ ] All character references use [character:X] placeholder syntax
- [ ] Timing conflict report attached
