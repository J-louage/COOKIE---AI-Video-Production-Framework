# Validate Script Checklist

## Required Checks
- [ ] User script received and input format identified
- [ ] All identifiable elements parsed and mapped to canonical fields
- [ ] Script normalized to canonical template format
- [ ] Header block present with title, episode ID, format type, target duration
- [ ] Character list present in header
- [ ] Every scene has a unique sequential scene ID
- [ ] Every scene has duration, setting, characters, visual direction, audio direction
- [ ] All character names resolved against characters directory
- [ ] Unresolved character names flagged for user clarification
- [ ] Character references converted to [character:X] placeholder syntax

## Quality Standards
- [ ] Timing is feasible for all scenes based on narration word count
- [ ] Narration fits within allocated duration at standard speech rates
- [ ] Total duration matches format preset constraints
- [ ] No critical ambiguities in visual directions
- [ ] No critical ambiguities in audio directions
- [ ] Camera movements specified for all visual scenes
- [ ] Music and SFX directions are actionable

## Output Verification
- [ ] Normalized canonical script produced
- [ ] Validation report generated with section-by-section analysis
- [ ] Ambiguity report included with improvement suggestions
- [ ] Overall readiness score calculated
- [ ] All original user intent preserved during normalization
