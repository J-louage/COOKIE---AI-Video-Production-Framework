# Quick Social Checklist

## Concept Input (Interaction Point 1)
- [ ] Social clip concept received from user
- [ ] Target platform determined (TikTok, Reels, Shorts)
- [ ] Format auto-set: 9:16, social-short preset
- [ ] Duration set (15-60 seconds, default: 30s)

## Micro-Script (Automated)
- [ ] Micro-script generated (3-6 scenes max)
- [ ] Hook present in first 1.5 seconds
- [ ] CTA or payoff at end
- [ ] Fast pacing (each scene 4-8 seconds)
- [ ] Narration text generated if beneficial
- [ ] Total duration within 15-60 second target

## Social SSD (Automated)
- [ ] All scenes set to 9:16 aspect ratio
- [ ] VEO resolution set to 720p
- [ ] TikTok-style captions enabled for all scenes
- [ ] Fast transitions between scenes
- [ ] Narration added to audio config (if applicable)
- [ ] Negative prompt optimized for mobile content

## Cost Approval (Interaction Point 2)
- [ ] VEO call count calculated (3-6 scenes)
- [ ] ElevenLabs cost included (if narration)
- [ ] 10% buffer applied
- [ ] User approved cost (or auto-approved in YOLO mode)

## Asset Generation (Automated)
- [ ] All VEO clips generated at 9:16 aspect ratio
- [ ] Narration generated via ElevenLabs (if applicable)
- [ ] Parallel generation used where possible
- [ ] Each generation cost logged to actuals
- [ ] VEO native audio used for ambient sound

## Social Composition (Automated)
- [ ] TikTok-style captions added (word-by-word highlight)
- [ ] Captions synced to narration or scene text
- [ ] Hook scene visually compelling in first 1.5 seconds
- [ ] CTA text overlay at end
- [ ] Fast cuts between scenes
- [ ] Narration audio ducking applied (if narration present)

## Render and Delivery (Interaction Point 3)
- [ ] Clip rendered at 720p in 9:16 format
- [ ] Platform-appropriate encoding preset used
- [ ] Aspect ratio verified: 9:16
- [ ] Duration verified: 15-60 seconds
- [ ] Captions present and readable
- [ ] Hook present in first 1.5 seconds
- [ ] Total cost logged and displayed
- [ ] Final clip presented to user
