# Episode Status Checklist

## Episode Data Loading
- [ ] Episode config loaded from episodes/{episode_id}/episode-config.yaml
- [ ] Production state block extracted (13 boolean flags)
- [ ] Episode status field read (not-started through complete)

## Asset Inventory
- [ ] Episode assets directory scanned
- [ ] Video clips counted vs total scenes in SSD
- [ ] Narration files counted vs scenes requiring narration
- [ ] Screen recordings counted vs screen-recording scenes

## Cost Summary
- [ ] Cost estimate loaded (if available)
- [ ] Cost actuals loaded (if available)
- [ ] Current spend calculated
- [ ] Percentage of estimate used calculated

## Episode Status Display
- [ ] Episode header shown (ID, title)
- [ ] Overall status displayed
- [ ] All 13 production state items shown with status icons
- [ ] Current phase identified from production state
- [ ] Asset summary shown (generated/total)
- [ ] Cost summary shown (spent/estimated with percentage)

## Project Overview (if applicable)
- [ ] All registered episodes listed
- [ ] Per-episode status and cost shown
- [ ] Project budget utilization displayed
- [ ] Total project spend calculated across all episodes
