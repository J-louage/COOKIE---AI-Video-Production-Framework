# Asset QA Checklist

For each scene:
- [ ] Video clip exists at expected path
- [ ] Video duration matches SSD specification (within 0.5s tolerance)
- [ ] Video resolution matches resolution profile
- [ ] Character appearance is consistent with reference images
- [ ] Camera movement matches SSD direction
- [ ] Lighting/mood matches style guide
- [ ] No visual artifacts (distortion, extra limbs, text corruption)
- [ ] Audio track present (if VEO native audio enabled)
- [ ] Narration generated and matches script text
- [ ] Narration timing matches SSD specification
- [ ] Multi-clip scenes properly merged with continuity
- [ ] First/last frame continuity verified across merged clips
