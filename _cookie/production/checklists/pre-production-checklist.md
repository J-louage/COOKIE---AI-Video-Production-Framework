# Pre-Production Checklist

- [ ] Canonical script finalized and reviewed
- [ ] All characters have identity.json files
- [ ] All characters have multi-angle reference images (front, side, back, 3/4)
- [ ] Style guide created and approved
- [ ] SSD (scene-spec.yaml) generated from script
- [ ] SSD validated against tool constraints (`/cookie-validate-ssd`)
- [ ] Asset dependency graph built
- [ ] Prompt library generated with consistency validation
- [ ] Cost estimate calculated and approved
- [ ] All character sheets validated (`/cookie-validate-char`)
- [ ] Resolution profile confirmed
- [ ] Export formats confirmed
