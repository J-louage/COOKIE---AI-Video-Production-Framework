# create-cookie-framework

Create a new [COOKIE AI Video Production Framework](https://github.com/J-louage/COOKIE---AI-Video-Production-Framework) project.

## Usage

```bash
npx create-cookie-framework my-project
cd my-project
claude
```

This will:

1. Clone the COOKIE framework into a new directory
2. Run the interactive setup script (dependency checks, API key configuration)
3. Leave you ready to open Claude Code and start producing videos

## What is COOKIE?

COOKIE is an AI agent framework for Claude Code that orchestrates end-to-end video production. It coordinates 13 specialized agents through 38 workflows to handle everything from initial concept to final rendered video — using VEO, Gemini, ElevenLabs, Remotion, and FFmpeg.

For full documentation, see the [main repository](https://github.com/J-louage/COOKIE---AI-Video-Production-Framework).
